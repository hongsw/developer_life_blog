Docker 를 이용한 워드프레스 개발 환경

프로젝트 마다 설정해주기 복잡하죠?

필수 개발 하드웨어 환경!

  1. MAC 구입한다.

  2. Windows PC는 버린다.

이하 자세한 설명은 정신건강을 위해서 생략한다.

개발 소프트웨어 환경!

  1. 개발가상환경을 구축하기 위해서 docker 를 설치 한다. https://www.docker.com/docker-mac

  2. mysql 접속을 위한 클라이언트로 sequelpro를 설치한다. http://www.sequelpro.com/

  3. 개발 에디터로 atom 를 사용한다. https://atom.io/

개발 폴더를 설정하고 개발 소스를 넣어서 가상환경에서 돌려보자 ~!

cd ~ && cd Github
mkdir myproject && cd myproject
git init
unzip myproject.zip

개발 폴더에 기존 레거시 스파게티 코드를 넣어주고 가상환경을 실행하자

모두 잘 알고 있듯이 워드프레스는 mysql, php이 기본적으로 필요합니다.

version: "2"
services:
  my-wpdb:
    image: mariadb
    volumes:
      - ./mysql:/var/lib/mysql 
    ports:
      - "8081:3306"
    environment:
      MYSQL_ROOT_PASSWORD: ChangeMeIfYouWant
  my-wp:
    image: wordpress
    volumes:
      - ./:/var/www/html
    ports:
      - "8088:80"
    links:
      - my-wpdb:mysql
    environment:
      WORDPRESS_DB_PASSWORD: ChangeMeIfYouWant

위의 내용을 docker-compose.yml 파일로 만들고 docker-compose up -d명령을 실행해서 나만의 개발환경을 시작하자( 처음에는 시간이 걸린다. )

http://localhost:8088/

이화면이 나온거라면 성공이다

기존 데이터베이스를 개발 디비에 접속해서 복원(import)하자

개발은 도커컴포저로 멋있게 시작하자 (쉽게 , 고생안하고)
