Elixir regex — http, email, img, tel, address

Elixre http://www.elixre.uk/

  1. url

    title = “hello http://medium.com/?test=11 12”

    #
    regex_url = ~r/(https?:\/\/)?(([\da-z.-]+).([a-z.]{2,6}))([\/\w .-])\/?(\&?)(([?])([^ \t\r\n\v\f])*)/

    첫번째 매칭되는 첫번째 문장

    link = Regex.run(regex_url, title) |> hd

    Replace matches in a string

    title = Regex.replace(regex_url, title, “”)

2. email (작성중)

title = "hello abc@abc.com bye"

3. img (작성중)

title = "hello http://ab.com/logo.png bye"

4. tel (작성중)

title = "hello +82-10-9999-1111 bye"

5. address (작성중)

etc

원인은 모르겠지만 parameter 부분에

[-a-zA-Z0–9:%_+.~#?&//=]를 넣으면 구문에러가 발생한다.

[^ \t\r\n\v\f]으로 변경하여 처리하였다.

*# A regex sigil to match 'foo'*
~r/foo/

*# Interpolation can be used*
~r/foo/ **=** ~r/#{"foo"}/

*# Test if a string matches*
true **=** Regex**.**match?( ~r/foo/ , "Hello foo")

*# Run returns first match*
["foo1", "1"] **=** Regex**.**run(~r/foo([0-9])/, "foo1 foo2")

*# Scan returns all matches*
[["foo1", "1"], ["foo2", "2"]] **=** Regex**.**scan(~r/foo([0-9])/, "foo1 foo2")

*# Replace matches in a string*
"one_two_three" **=** Regex**.**replace(~r/-/, "one-two-three", "_")
