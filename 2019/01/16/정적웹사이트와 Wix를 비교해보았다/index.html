정적웹사이트와 Wix를 비교해보았다.

우선 팩트폭행이 트랜드.

예시를 보자

http://www.collegecommentary.net/

이 사트는 wix.com으로 만든 이쁜사이트이다.

  • 빠르게 만들수 있다.

  • 간단한 기능(예약)도 바로 사용가능하다.

하지만 속도를 보라

  • 처음 접속한사람(브라우저 캐시를 끄고)은 1.3초부터 화면이 보이기 시작하고 20.72초에 화면이 완성된다.

  • 반응형이 아니라서 모바일에서는 redirect되는 시간이 추가된다.

그러면 Static Web Hosting은 어떻까?

자 static web hosting에서 gitpage 를 빼놓을 수 없다.

거기에 bootstrap으로 만들어서 (반응형)이 되도록 다시 만들어서 올리자

https://collegecommentarynet.github.io/collegecommentary.net/

  • 처음 접속한사람(브라우저 캐시를 끄고)은 1.18초부터 화면이 보이기 시작하고 1.41초에 화면이 완성된다. 무려 19.31초 빨리 화면이 완성된다.

  • 반응형이라서 모바일에서는 redirect되는 시간이 없다.

아직 도메인은 바꾸지 않은 상태이다.

그전상태와 비교를 해보고 싶다.
